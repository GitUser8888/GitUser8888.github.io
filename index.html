Project Highlights

Face Value: Using AI to Predict Stardom in South Korean Drama Industry

In a short span of 2 weeks, I built an image classification machine learning model that uses face image to predict potential popularity or success of the subject in the South Korean drama industry (males).

The models are trained with the face images from 490 South Korean male actors collected from an active community-based drama database site. The male actors are divided into 2 categories, “very popular” and “not so popular”.

Check out the demo video, repo, and Streamlit demo.

Classification of Bipolarism and Schizophrenia Subreddits using NLP

In another short span of 2 weeks, using Natural Language Processing (NLP), I built a model that classifies if a reddit post belongs to r/schizophrenia or r/bipolar. This can be further developed into a linguistic tool that helps clinicians better understand how mental health can be diagnosed via non-contextual linguistic information.

Check out the repo, and Streamlit demo.
